# Dragon news app

## Some project features
- REST API
- News JSON data
- Category JSON data 
- Responsive
- Category Wish news display
- Firebase authentication (Google, Facebook)
- Firebase Auth (email and password)
- Random news display in left sidebar
- News Details page
- Private Route for user
- Context API
- Custom Hooks
- Custom Provider
- PropTypes
- Tailwindcss
